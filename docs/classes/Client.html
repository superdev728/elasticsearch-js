Client Class

Defined in: src/lib/client.js:6

A client that makes requests to Elasticsearch via a Transport

Initializing a client might look something like:

var client = new Elasticsearch.Client({
  transport: {
    hosts: [
      'es1.net:9200',
      'es2.net:9200'
    ],
    sniff_on_start: true
  },
  log: {
    type: 'file',
    level: 'warning'
  }
})
